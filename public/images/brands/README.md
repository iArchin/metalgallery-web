# Brand logos

Drop each brand's official logo here using the **exact** filename below. As soon
as a file exists it appears in the home "برندهای محبوب" marquee; until then a
styled brand-name wordmark is shown instead. No code or admin change is needed.

| Brand (نام) | File to add |
| --- | --- |
| لگو (LEGO) | `lego.png` |
| متل (Mattel) | `mattel.png` |
| هاسبرو (Hasbro) | `hasbro.png` |
| فیشر پرایس (Fisher-Price) | `fisher-price.png` |
| پلی‌موبیل (Playmobil) | `playmobil.png` |
| بندای (Bandai) | `bandai.png` |

Tips:
- Prefer **PNG or SVG with a transparent background**; wide wordmark logos look
  best (they're shown with `object-contain`, so they won't be cropped).
- Using a different name or extension? Edit the brand's **لوگو برند** field in
  the admin panel (برندها → ویرایش) to point at the new path — e.g.
  `/images/brands/lego.svg`.
- New brands you add in the panel can reference a file here the same way.
